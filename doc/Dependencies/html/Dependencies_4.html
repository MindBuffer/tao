Tools used to build the programs and libraries

Tao uses the GNU tools autoconf, automake and libtool for compilation and installation. You must have these packages installed if you intend to build Tao from the sources. autoconf also relies upon the macro language GNU m4 so you'll need that too! All of these packages are available from http://www.gnu.org.
