Mouse bindings


Key bindings

Down-arrow
Reduce the visible amplitude of the vibrations.
Up-arrow
Increase the visible amplitude of the vibrations.
Left-arrow
Decrease the number of ticks between displayed frames i.e. make the animation slower but smoother.
Right-arrow
Increase the number of ticks between displayed frames i.e. make the animation faster but more jerky.
Esc
Exit and close the graphics window.
