Using the Hammer device - hammer.tao

This script illustrates the use of the Hammer device. It creates a single string and a single hammer. It then locks the ends of the string, applies the hammer to a point x=0.7 on the string, and sets some hammer attributes.

Finally it drops the hammer at the beginning of the performance, leaving it to bounce naturally on the string. 

    Audio rate: 44100;
        
    String string(200 Hz, 30 secs);
        
    Hammer hammer;
        
    Init:
        string.lockEnds();
        string(0.7) -- hammer;
        hammer.setGravity(0.0001).setMass(200.0);
        ...
        
    Score 10 secs:
        At 0 secs:
            hammer.drop();
            ...
        ...
