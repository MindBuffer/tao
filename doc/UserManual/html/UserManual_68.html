For Statement

The For statement provides a rudimentary looping and iteration facility. It is fairly basic as it only supports integer counting from some initial value to some final value in steps of 1. An example is given below: 

    String stringArray[4]=
        {
        (50 Hz, 60 secs), 
        (100 Hz, 60 secs), 
        (150 Hz, 60 secs), 
        (200 Hz, 60 secs)
        };
     
    Counter c;
     Init:
        For c=0 to 3:
            stringArray[c].lockEnds();
            ...
        ...
