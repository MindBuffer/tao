Pitch Declarations

We have already come across the use of pitch literals in section * but it is also possible to declare pitches as seperate objects in a script. Pitch declarations take the following form: 

    Pitch p1=C#7+1/2, p2=8.3 oct, p3=8.01 pch, p4=50 Hz;

This example illustrates the four pitch formats supported, i.e. note name, octave/fraction, octave/semitone and frequency.
