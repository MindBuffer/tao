Building and Installing Tao

Tao was initially developed on SGI Irix 5.3 but was ported in 1998 to Red Hat Linux 5.0. Since then it has been developed primarily on Red Hat 6.0. It has not been tested with other Unix systems but since it only uses widely available Unix tools and platform independent libraries it is highly probable that it would work just as well on other systems.
